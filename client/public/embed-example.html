Artwork Uploader Button - Integration Guide

Quick Start

Add the Artwork Uploader "Order Transfers" button to any webpage with just one line of code:

<script src="https://yoursite.com/embed-button.js"></script>

The button will automatically appear where you place the script tag.

Method 1: Automatic Button

Add a container with the data-artwork-uploader attribute:

<div data-artwork-uploader></div> <script src="embed-button.js"></script>

Demo:

Method 2: Using Container ID

Create a container with id="artwork-uploader-button":

<div id="artwork-uploader-button"></div> <script src="embed-button.js"></script>

Demo:

Method 3: Programmatic Creation

Create buttons dynamically using JavaScript:

<script src="embed-button.js"></script> <script> // Create button in a specific container const container = document.getElementById('my-container'); ArtworkUploader.createButton(container); // Or open directly ArtworkUploader.open(); </script>

Configuration Options

Customize the button by setting global variables before loading the script:

<script> // Set configuration before loading the script window.ARTWORK_UPLOADER_URL = 'https://yoursite.com/artwork'; window.ARTWORK_UPLOADER_BUTTON_TEXT = 'Design Your Transfer'; window.ARTWORK_UPLOADER_BUTTON_CLASS = 'custom-style-button'; window.ARTWORK_UPLOADER_OPEN_MODE = 'popup'; // or 'redirect' </script> <script src="embed-button.js"></script>

Configuration Parameters:

  • ARTWORK_UPLOADER_URL - The URL of your artwork uploader app (default: current origin)
  • ARTWORK_UPLOADER_BUTTON_TEXT - Button text (default: "Order Transfers")
  • ARTWORK_UPLOADER_BUTTON_CLASS - CSS class for custom styling
  • ARTWORK_UPLOADER_OPEN_MODE - "popup" or "redirect" (default: "popup")

Advanced Usage

Multiple Buttons with Different Configurations

<!-- First button --> <div id="button1"></div> <!-- Second button --> <div id="button2"></div> <script src="embed-button.js"></script> <script> // Create first button const btn1 = ArtworkUploader.createButton(document.getElementById('button1')); btn1.textContent = 'Design T-Shirt'; // Create second button const btn2 = ArtworkUploader.createButton(document.getElementById('button2')); btn2.textContent = 'Design Hoodie'; btn2.className = 'custom-hoodie-button'; </script>

Troubleshooting

Button Not Appearing?

  • Ensure the script is loaded after the DOM elements
  • Check browser console for JavaScript errors
  • Verify the script URL is correct and accessible

Popup Blocked?

Modern browsers may block popups. Consider using redirect mode for better compatibility:

window.ARTWORK_UPLOADER_OPEN_MODE = 'redirect';
